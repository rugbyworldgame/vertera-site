# vertera-site

Статический сайт партнёра Vertera на GitHub Pages.  
Дизайн: тёмный “dark emerald”, стеклянные карточки, аккуратная мобильная навигация.

## Запуск
Это статический сайт. Достаточно открыть `index.html` в браузере.

Если сайт опубликован на GitHub Pages — он открывается по ссылке Pages (в настройках репозитория).
